# builds/

Shared build-output directory for every Makefile in this repo. Each
subproject's Makefile places its compiled binary here (e.g.
`dbj_email_crud.exe`), rather than in its own source folder.

This whole directory is gitignored (`/builds/` in the repo root
`.gitignore`) — nothing here is checked in. Contents are disposable
build artifacts and per-machine local run config (e.g. `.ini` files),
regenerated by running the relevant Makefile.

`my.ini` in this directory is a working config file — do not erase it.
